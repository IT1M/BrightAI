## أسئلة سريعة قبل الإقفال النهائي
هذه الأسئلة ستُحسّن دقة الخطة والتوصيات. يمكنك الرد عليها لاحقًا، والتقرير أدناه مبني على الواقع الحالي في المشروع.
- ما الخدمات الأعلى ربحية والأولوية (استشارات، أتمتة، وكلاء AI، أرشفة طبية، تحليل بيانات…)?
- ما المدن المستهدفة أولًا داخل السعودية (الرياض/جدة/الدمام/الخبر/مكة/المدينة)؟
- ما هدف التحويل الأساسي: حجز استشارة، طلب عرض سعر، تسجيل تجربة، أم مكالمة مباشرة؟

## 1) ملخص تنفيذي مختصر
- البنية الحالية غنية بالمحتوى، لكن هناك مشاكل تقنية تؤثر مباشرة على الفهرسة، تجربة المستخدم، وظهور النتائج.
- أكبر العوائق: روابط وسياقات URL غير نظيفة، تكرار محتوى بين مسارات مختلفة، ملفات CSS مفقودة في عدد كبير من الصفحات، وصور OG/PWA غير صحيحة.
- فرصة التصدر عالية إذا تم توحيد البنية، إصلاح الموارد الحرجة، وإعادة بناء التجميع الموضوعي (Content Clusters) حسب السوق السعودي.

## 6) تحسينات SEO عملية حسب المحور

### 6.1 البنية (Structure)
- تحويل البنية إلى مسارات نظيفة بدون `/frontend/pages/` في الـ URL العام.
- اعتماد هيكل Hub/Spoke:
  - `/solutions/` كصفحة محور.
  - صفحات تفصيلية لكل قطاع.

### 6.2 المحتوى (Content)
- إنشاء صفحات Pillar لكل خدمة رئيسية.
- إضافة دراسات حالة سعودية حقيقية.
- تضمين عناصر Trust محلية (NCA/NDMO/رؤية 2030) داخل المحتوى.

### 6.3 HTML (On-Page)
- عناوين فريدة 100% لكل صفحة.
- وصف ميتا بين 140–165 حرفًا.
- H1 واحد + تقسيم H2/H3 واضح.
- إضافة Breadcrumbs لكل صفحة داخلية.

### 6.4 الأداء (Performance)
- إلغاء Tailwind CDN واعتماد CSS مدمج ومُسبق البناء.
- ضغط الصور (WebP/AVIF) + تحديد `width/height`.
- إزالة أي مكتبات لا تُستخدم في أول تحميل.

### 6.5 تجربة المستخدم (UX)
- تقليل وقت الوصول لأزرار CTA فوق الطية.
- إضافة إثبات اجتماعي وصيغ ضمان.
- تبسيط نماذج التحويل (اسم + جوال + مدينة كحد أدنى).

## 12) أمثلة كود سريعة (قابلة للنسخ)

### مثال Canonical + Hreflang
```html
<link rel="canonical" href="https://brightai.site/solutions/automation/">
<link rel="alternate" hreflang="ar-SA" href="https://brightai.site/solutions/automation/">
<link rel="alternate" hreflang="en" href="https://brightai.site/en/solutions/automation/">
```

### مثال Lazy Loading للصور
```html
<img src="/assets/images/hero.webp" alt="حلول الأتمتة الذكية" width="1200" height="630" loading="lazy" decoding="async">
```

### مثال JSON-LD لخدمة واحدة مطابقة للمحتوى
```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Service",
  "name": "الأتمتة الذكية للشركات السعودية",
  "provider": { "@type": "Organization", "name": "Bright AI" },
  "areaServed": "SA",
  "serviceType": "Process Automation"
}
</script>
```
